1. 产品定位

BioRender 是面向生命科学科研场景的在线可视化 SaaS，核心价值是把科学准确的素材库、可视化画布、图表统计、AI 辅助、团队协作、出版导出授权整合到一个工作流中。

科研插图论文图、机制图、图形摘要、Grant application 配图。
展示材料学术海报、会议展示、Slides、多页故事结构。
实验流程Protocol、timeline、flowchart、SOP 可视化。
数据图表原始数据导入、图表创建、统计分析、图表嵌入。
说明：app.biorender.com 是需要 JavaScript 和登录态的 SPA，本报告仅保留可由 BioRender 官方帮助中心和产品页核验的功能点。

3. 模板库功能清单

BioRender 的模板库不只是画布内的模板搜索面板，而是一个独立的内容发现、模板分发和社区贡献体系。它面向“先找现成科研图，再进入画布改图”的工作流。

3.1 模板库页面结构

  • 顶部导航：Gallery、BioRender Templates、Learning Hub、Upgrade、通知、账号入口。
  • 左侧 Discover：Illustrations、Posters、Creators，用于按内容类型发现模板。
  • 左侧 Browse：All、Community、Liked by me，用于浏览全部模板、社区模板和个人收藏。
  • 左侧 Categories：按科研主题分类，例如 AI & Drug Discovery、Top Disease Mechanisms、Activity Sheets、Agriculture & Plants、Anatomy & Clinical、Cancer、Chemistry、Graphs & Timelines、Icon Packs 等。
  • 主内容区：模板搜索、提交模板、排序、下载/视图切换、卡片网格。
  • 专题区：Top Disease Mechanisms Library 提供按疾病领域组织的机制图模板，并带有领域筛选。

4. 画布编辑器功能清单

4.1 实际界面结构

  • 顶部导航：菜单、BioRender 标识、文件名、云端保存状态、撤销/重做。
  • 顶部中间：Edit canvas / 编辑画布入口。

5. Graphing 绘图功能清单

BioRender Graphing 是独立于普通画布的科研数据绘图与统计分析工作区。官方文档把它定义为从数据导入、自动数据整理、图表创建、异常值检测到统计分析的一体化流程；Graphing 文件以 Data / Graph / Analysis 三段式工作区组织数据、图表和分析结果。

5.1 官方工作流与界面结构

  • 创建入口：在 Gallery 中点击 Create New，选择 Graph，再点击 New Graphing 创建 Graphing 文件。
  • 顶部结构：左侧为 Graphing 品牌、文件名、保存状态、撤销/重做；中间是 Data / Graph / Analysis tab；右侧是 Share 和 Export。
  • Data tab 空状态：左侧 Variables 面板显示 No variables yet，提示上传或输入数据后点击 Detect variables；主区域显示 No datasets yet，支持 Blank spreadsheet 和 Upload dataset。
  • 数据导入：支持 Smart Data Import 上传 .csv/.xls/.xlsx/.prism/.pzfx，最高 50MB；多 sheet 文件可选择全部或部分 sheet；Prism 文件默认可导入 graphs 和 analyses，如果只要原始数据可取消该选项。
  • 空白表格：可从 Blank spreadsheet 开始，复制粘贴数据，右键增删行列，完成后点击 Re-detect variables；系统会复用 Smart Data Import 流程识别变量。
  • 变量识别：系统自动识别变量、处理合并表头、忽略主数据集外的机器注释、忽略空行空列；用户可编辑变量名、变量分配、变量覆盖的高亮单元格，也可增删变量。
  • 数据集组织：导入后每个 dataset 作为 Data tab 底部 sheet/dataset tab 出现；左侧列出当前 dataset 的变量；数据表按变量用不同颜色框选列，变量卡展示变量名、示例值或数值范围。
  • Data tab 已有数据状态：顶部显示 Datasets(n)，右上支持设置、Add dataset 和 Graphs；右侧 Graphs 面板展示与当前 dataset 关联的图表，支持 Create new、View all 和图表卡片预览。
  • Graph tab：左侧 Graph details 配置 dataset、filter、graph type、X-axis variable、Y-axis variable、grouping variable；中间是 Graph preview 和 Displayed data；右侧是 Customize graph，在点击 Continue 前显示 Editing in progress。
  • 图表创建：从 Data tab 的 Create new graph 或 Graph tab 创建图表；选择图表类型，例如 bar/column/scatter/mean/median line，指定变量后预览实时更新，确认后点击 Continue 打开右侧自定义面板。
  • 图表自定义：Continue 后可调整 Display data、颜色和字体、轴格式、间距、线条等设置；graph details 可重新打开编辑，未保存修改时需要确认保留或丢弃。
  • Analysis tab：用于运行和查看统计分析；Graph 编辑状态未确认时，Run analysis 会不可用或需要先确认图表 details。
  • 异常值与统计：Graphing 会对每个 graph 自动运行 ROUT 异常值检测；统计分析会基于预检和假设检验推荐合适方法，并在结果中展示表格、描述性统计和显著性标记。
  • 多图管理：Graph Grid View 可按名称搜索、按 dataset 筛选、按名称/创建时间/修改时间排序，并支持 duplicate、style match、delete 等批量操作。

6. Poster Builder 功能清单

BioRender Poster Builder 更接近“结构化海报排版器”，而不是通用自由画布。官方文档把它的核心能力放在模板/主题、左侧 Insert Content 面板、版块容器、图片与标题说明、间距控制、字号规范和导出交付上。

6.1 官方工作流与界面结构

  • 创建入口：在 Gallery 视图点击 Create New File，选择 Poster Builder。
  • 起稿方式：进入 Poster Builder 后选择预制 scientific poster layouts 和 color theme；如果不使用模板，可关闭模板选择窗口，从空白海报开始。
  • 模板覆盖：官方模板覆盖 conference presentations、research posters、lab reports 等海报布局。
  • 模板搜索：BioRender Templates 支持按关键词或作者搜索模板；模板也可来自 BioRender 官方团队和社区贡献者。
  • 模板使用：模板可作为新文件起点，进入编辑后仍可继续调整海报尺寸、颜色、字体、section placements 等。
  • 搭建方式：通过左侧 Insert Content 面板拖拽内容块到海报，官方明确包括 Title、Section、Text、Image、Header、Padding、Caption 等。
  • 标题区：每张海报默认带 Title；标题可移动，内部内容会一起移动；标题横跨整张海报宽度或高度时，背景填充会延伸到海报边缘；Logo 可通过 Image block 放入标题区域。
  • Section 容器：Section 用来组织 Introduction、Methods、Results、Conclusion、Acknowledgements 等内容；移动 Section 时，内部文本、图片和 header 会一起移动；Section 会自动适配移动后的内容，避免内容越出画布。
  • 文本与字号：文本框内容增加时不会自动撑大，需要拖动蓝色边线调整尺寸；48 x 36 in 海报的建议字号为 Title 96pt 以上、Section headers 48pt 以上、Body text 28-36pt、Figure captions 18-28pt、References/次要文本 18-22pt。
  • 图片与图表：Image block 可插入 BioRender illustration，也可上传 PNG/JPG/SVG 图片或 Logo；Graph 文件可从 Gallery 选择并插入图片区域，在海报内做视觉样式调整，但不影响原 Graph 文件和底层数据集。
  • 交付约束：Poster Builder 导出的海报都会带 Created with BioRender Poster Builder 水印；Basic 支持 PNG/JPG/PDF 的 72 DPI 导出，Premium 支持 PNG/JPG/PDF 的 72、150、300 DPI。
